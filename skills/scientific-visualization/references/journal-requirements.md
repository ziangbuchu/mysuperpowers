# Journal Defaults

These are defaults for helper scripts, not a substitute for checking the target venue.

## Widths

- Nature: single 89 mm, double 183 mm
- Science: single 55 mm, double 175 mm
- Cell: single 85 mm, double 178 mm
- PLOS: single 83 mm, double 173 mm

## Export Hints

- Line art: 600 to 1000 DPI or vector output
- Photos: 300 DPI
- Mixed figures: 300 to 600 DPI

## Workflow Recommendation

For normal result-analysis artifacts in this repository:

- export `png` and `pdf`
- use Nature-like single-column sizing unless a multi-panel layout clearly needs double-column width
- only tighten journal-specific export settings when the user explicitly asks for submission-ready output
